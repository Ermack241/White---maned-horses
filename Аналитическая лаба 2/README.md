## Цель работы

|<a name="_hlk155962199"></a>Azure App Center|Сервис, предназначенный для сборки, тестирования, распространения и мониторинга мобильных приложений.|

|Azure App Service|Сервис облачных вычислений, предназначенный для развертывания, масштабирования и управления веб-приложениями и API, обеспечивает простой способ создания, развертывания и обслуживания веб-приложений без необходимости управления инфраструктурой.|
|Azure Monitor|Сервис облачного мониторинга и управления, предназначенный для обеспечения централизованного мониторинга, диагностики и аналитики состояния ресурсов в облачной среде Azure.|
|Azure Automation|Облачная служба автоматизации, обновления операционной системы и настройки, которая поддерживает согласованное управление в среде Azure и за её пределами. Она включает автоматизацию процессов, управление конфигурацией, управление обновлениями и общие возможности.|
|Azure SQL servers|Служба облачных баз данных, предоставляющая управляемые реляционные базы данных на основе СУБД Microsoft SQL Server, предоставляет возможность хранения, управления и масштабирования баз данных без необходимости управления физической инфраструктурой.|
|Azure Bastion|Сервис, предназначенный для обеспечения безопасного и защищенного удаленного доступа к VM в виртуальной сети, обеспечивает удобный механизм для подключения к виртуальным машинам через веб-браузер, минимизируя уязвимости, связанные с прямым открытием портов RDP (Remote Desktop Protocol) или SSH (Secure Shell).|
|Azure Data Factory|Служба, предназначенная для создания, запуска и управления комплексными потоками данных между различными источниками данных. Это может включать в себя передачу данных из баз данных, файловых хранилищ, приложений и других источников.|
|Azure Firewall|Сервис, предназначенный для обеспечения безопасности и контроля доступа к сетевым ресурсам в облаке, предоставляет высокоуровневые средства управления трафиком, а также защиту сети от угроз и вредоносных атак.|
|Azure Power BI|Облачный сервис, предназначенный для визуализации данных и создания интерактивных отчетов и панелей управления, предоставляет возможность собирать, анализировать и представлять данные в удобной форме.|
|Azure Site Recovery|Служба, предназначенная для создания резервных копий и восстановления виртуальных машин и физических серверов, обеспечивает возможность восстановления бизнес-приложений и рабочих нагрузок в случае сбоев.|
|Azure Virtual Machine|Виртуальный компьютер, который предоставляет гибкую и масштабируемую вычислительную мощность в облаке, позволяя пользователям развертывать и управлять виртуальными машинами с различными конфигурациями ресурсов и операционными системами.|


4.	Определение соответствия каждого сервиса международного провайдера русскому сервису

Azure App Center - Yandex Cloud Apps, приложения VK Cloud

Azure App Configuration - Yandex Cloud Apps, приложения VK Cloud

Azure App Service - Yandex Cloud Apps, приложения VK Cloud

Azure Monitor - Yandex Monitoring (позволяет собирать и хранить метрики, а также отображать их в виде графиков на дашбордах); Cloud Monitoring от VK Cloud (сбор нужных данных о системе, в том числе любые пользовательские параметры)

Azure Automation - Yandex Resource Manager (определяет ресурсную модель Yandex Cloud и позволяет структурировать ресурсы с помощью каталогов)

Azure SQL servers - Yandex Managed Service for MySQL (управление реляционной СУБД в инфраструктуре Yandex Cloud); DBaaS от VK Cloud (полностью управляемые и масштабируемые СУБД с гарантированным уровнем сервиса)

Azure Bastion - Yandex Virtual Private Cloud (управление облачными сетями, которые передают информацию между ресурсами Yandex Cloud и обеспечивают им доступ в интернет); Private Cloud от VK Cloud (платформа для построения частного облака в ЦОДе заказчика. Включает набор готовых ИТ-инструментов для разработки и безопасности и маркетплейс бизнес-приложений)

Azure Data Factory – CDN от VK Cloud (доставку контента с минимальной задержкой от ваших серверов к пользователям даже во время пиковых нагрузок)

Azure Firewall - Yandex Virtual Private Cloud (управление облачными сетями, которые передают информацию между ресурсами Yandex Cloud и обеспечивают им доступ в интернет)

Azure Power BI - Yandex DataLens (визуализации и анализа данных); Cloud Spark от VK Cloud (обработка данных любого объема)

Azure Site Recovery - Yandex Cloud Backup (создание резервных копий и восстановления виртуальных машин); Cloud Backup от VK Cloud (автоматизированное управление резервными копиями)

Azure Virtual Machine - Yandex Compute Cloud (предоставление масштабируемых вычислительных мощностей для размещения, тестирования и прототипирования проектов)

## Вывод
В данной лабораторной работе наша команда познакомилась с облачными сервисами Azure, которые предлагают огромный спектр функций. Выяснилось, что их можно заменить сервисами от Yandex Cloud или VK Cloud, но они либо недостаточны полны по функциалу, либо сразу объединяют функции, которые предоставляются отдельными сервисами Azure. Таким образом миграция возможна, но только если пользователя будут устраивать данные недостатки.
